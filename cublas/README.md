# Cublas_LU

CuBLAS LU decomposition code from [NVIDIA repository](https://github.com/NVIDIA/cuda-samples)

## Build and Run (Linux)

### Build
```bash
make TARGET_ARCH=x86_64 SMS="75" # dbg=1 HOST_COMPILER=g++
```

### Run
```bash
make run # ./simpleCUBLAS_LU
```
